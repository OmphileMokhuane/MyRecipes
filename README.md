## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/recipe-book.git
cd recipe-book
```

2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
node server.js
```
4. Open your browser and navigate to:

```bash
http://localhost:3000
```
